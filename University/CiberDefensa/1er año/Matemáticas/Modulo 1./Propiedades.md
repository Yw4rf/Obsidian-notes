#### Orden de las operaciones
- Para evaluar correctamente las operaciones se debe seguir una secuencia específica:
	1. **Paréntesis o corchetes**
	2. **Exponente y raíces**
	3. **Multiplación y división (de izquierda a derecha)**
	4. **Suma y Resta (de izquierda a derecha)** 

#### Propiedades de Suma y Multiplicación
- **Propiedad Conmutativa**: 
	- El orden de los números no altera el resultado. En la suma $a+b = b+a$ y en la multiplicación $a \times b = b \times a$  
- **Propiedad Asociativa**: 
	- La forma en que se agrupan tres o más números (usando paréntesis) no cambia el resultado final. Para la suma es $(a + b) + c = a + (b + c)$ y para la multiplicación es $(a \times b) \times c = a \times (b \times c)$ 
- **Propiedad Distributiva**:
	- Multiplicar un mismo número por una suma es igual a multiplicar ese número por cada sumando y luego sumarlos. $c \times (a + b) = c \times a + c \times b$ 
- **Propiedad de Cierre**: 
	- Los números reales son **cerrados** bajo la suma y la multiplicación, lo que significa que el resultado de estas operaciones siempre será otro número real. También se aplica a la resta y a la división (excepto por cero)

#### Propiedades de la Igualdad 
- **Propiedad Simétrica**: 
	- Si $a = b$ entonces $b = a$
- **Propiedad Transitiva**: 
	- Si $a = b$ y $b = c$ entonces $c = a$
- **Propiedad Suma y Resta**: 
	- Si se añade o resta el mismo número a ambos lados de una ecuación, la igualdad se mantiene $a + c = b + c$
- **Propiedad de Multiplicación y División**: 
	- Si se multiplica o divide ambos lados de una ecuación por el mismo número (distinto de cero, la igualdad persiste. 

#### Propiedades de Valor Absoluto
- El valor absoluto de un número $x$, denotado como $|x|$, representa su distancia respecto al cero y siempre es un valor no negativo. Sus propiedades incluyen: 

- **Propiedad de Idempotencia:**
	- Tomar el valor absoluto más de una vez no cambia el resultado $||x|| = |x|$
- **Propiedad de Simetría**: 
	- El valor absoluto de un número y su opuesto son iguales $|-x| = |x|$ 
- **Propiedad Multiplicativa**: 
	- El valor absoluto de un producto es el producto de los valores absolutos $|x \times y| = |x| \times |y|$ 
- **Preservación de la división**: 
	- El valor absoluto de un cociente es el cociente de los valores absolutos. E.g: $|\frac{x}{y}| = \frac{|x}{y}|$
