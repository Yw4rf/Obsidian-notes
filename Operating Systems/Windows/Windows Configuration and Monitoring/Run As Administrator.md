#### Run As Administrator 
- No es recomendable logear en Windows usando una cuenta de Administrador o una cuenta con privilegios de Administrador. Esto debido a que cualquier programa que se ejecute en ese tipo de cuentas heredará los mismos privilegios.
- **Ejecutar un programa con privilegios de administrador otorga acceso ampliado al sistema**, permitiendo modificar archivos y carpetas protegidos, gestionar hardware con mayor control y realizar cambios en la configuración del sistema.
- Un programa malicioso con privilegios de administrador puede tomar control total del equipo, modificar o eliminar archivos críticos, y evadir controles de seguridad.
