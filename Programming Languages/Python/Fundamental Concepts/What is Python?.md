#### What is Python?
- Python es un lenguaje de programación en el cual el código se ejecuta por un intérprete linea a linea. Es de tipado dinámico, esto quiere decir que las variables no tienen un tipo fijo escrito, el valor tiene tipo y el nombre es un enlace a ese objeto. Es un lenguaje multiparadigma: soporta programación imperativa, funcional y orientada a objetos.
